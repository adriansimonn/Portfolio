# Portfolio Website

A modern, minimal portfolio website inspired by ZARA's clean aesthetic. Built with HTML, CSS, and JavaScript.

## Features

- **Minimal Design**: Clean, white background with elegant typography
- **Magazine Style**: Professional layout with ample whitespace
- **Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Easy Navigation**: Simple top navigation bar with clear page structure
- **Information-Rich Homepage**: Comprehensive about me section with professional details

## Pages

- **Home** (`index.html`): About me section with split layout (image and text)
- **Experience** (`experiences.html`): Professional experience and education
- **Work** (`work.html`): Projects, publications, and open source contributions
- **Blog** (`blog.html`): Blog posts and articles

## Customization

### Adding Your Photo

Replace the placeholder in `index.html`:
```html
<div class="image-placeholder">
    <img src="path/to/your/photo.jpg" alt="Your Name">
</div>
```

### Updating Content

1. **Personal Information**: Edit the text sections in `index.html`
2. **Experience**: Update `experiences.html` with your work history
3. **Projects**: Add your projects to `work.html`
4. **Blog Posts**: Update `blog.html` with your articles
5. **Contact Links**: Update email and social media links in `index.html`

### Styling

All styles are in `styles.css`. Key variables you can customize:
- `--primary-black`: Main text color
- `--primary-white`: Background color
- `--text-gray`: Secondary text color
- `--border-gray`: Border and divider color

## Getting Started

1. Open `index.html` in a web browser
2. Customize the content with your information
3. Add your photo to replace the placeholder
4. Update links and contact information
5. Deploy to your preferred hosting service
